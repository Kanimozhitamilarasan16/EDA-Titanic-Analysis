# Exploratory Data Analysis (EDA) on Titanic Dataset
INTERN ID : CITS5189
## Objective
To perform Exploratory Data Analysis (EDA) on the Titanic dataset using Python.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Titanic tested.csv dataset

## Steps Performed
- Imported libraries
- Loaded dataset
- Checked dataset information
- Handled missing values
- Removed duplicate records
- Performed Exploratory Data Analysis
- Created visualizations

## Visualizations
- Survival Count
- Gender Distribution
- Passenger Class Distribution
- Age Distribution
- Correlation Heatmap
- Age vs Survival Box Plot

## Conclusion
The dataset was cleaned and analyzed successfully using various visualization techniques to understand passenger demographics and survival patterns.
